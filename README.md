**混沌置乱的循环阶分析**

Analyzing the cycle structure of permutation tables generated by chaotic maps via `np.argsort`, evaluating their cryptographic security through Landau's theorem, precision degradation, seed avalanche effect, and sorting bias.

---

## Project Overview

Chaotic maps are widely used in image encryption to generate permutation tables (scrambling indices). This project systematically studies the **cycle decomposition** of such permutations — a fundamental property that determines the **order** (the number of iterations after which the permutation returns to identity).

### Key questions

- Do chaotic permutations approach the theoretical randomness bound (Landau's theorem)?
- Does finite-precision computation degrade security?
- Does the sorting operation (`argsort`) itself introduce structural bias?
- How sensitive are these permutations to initial conditions (seed avalanche)?

---

## Chaotic Maps (5 types)

| Map | Formula | Parameter | Domain |
|-----|---------|-----------|--------|
| Logistic | $x_{n+1} = \mu x_n(1-x_n)$ | $\mu = 3.99$ | $(0,1)$ |
| Tent | $x_{n+1} = r\min(x_n, 1-x_n)$ | $r = 1.99$ | $(0,1)$ |
| Chebyshev (k=3) | $T_3(x) = 4x^3 - 3x$ | $k = 3$ | $(-1,1)$ |
| Sine | $x_{n+1} = r\sin(\pi x_n)$ | $r = 0.99$ | $(0,1)$ |
| Henon | $(x,y) \to (1-ax^2+y, bx)$ | $a=1.4, b=0.3$ | $x\in(-1.5,1.5)$ |

All maps apply **M=1000 warmup** iterations before collecting N samples.

## Experiments

| ID | Experiment | Purpose |
|----|-----------|---------|
| E1 | Average $\ln(\text{order})$ vs N | Compare cycle properties across maps |
| E2 | Landau reference | $\mathbb{E}[\ln(\text{order})] \sim \sqrt{N\ln N/2}$ |
| E3 | Seed avalanche | DiffRate + Spearman Footrule |
| E4 | Parameter scan (optional) | Detect periodic windows |
| E5 | Cycle length distribution | N=1024, histogram per map |
| E6 | Precision degradation | float32 vs float64 vs Decimal(50) |
| E7 | Sorting bias | Chaos+argsort vs Random+argsort vs Fisher-Yates |
